# Release Notes

## v0.1.3 - 2026-07-18 (release preparation)

- Added aggregate PAXHR-PAXMIN IS concordance summaries and two-cycle BPA-IS estimates at hourly and minute resolution.
- Added aggregate questionnaire-sensitivity and IQR-scale descriptive effect-size summaries.
- Added an aggregate-only PAXMIN calibration summary script that regenerates a PDF/PNG/CSV from public result tables.
- Added build-time replacement of machine-specific project paths in copied CSV provenance fields with `<project-root>`.
- Raw PAXMIN XPT files, minute trajectories, participant-level matched pairs, and raw-scanning code remain excluded.
- Published as GitHub release `v0.1.3` and archived at Zenodo DOI https://doi.org/10.5281/zenodo.21429423; the `v0.1.2` DOI must not be cited for these additions.

## v0.1.2 - 2026-07-16

- Created a distinct archival snapshot after refreshing the Zenodo GitHub integration.
- Preserved the reviewed public scope: portable analysis code, nonrestricted aggregate results, provenance records, and public-data retrieval instructions.
- No analysis results, source data, submission-system figures, or formatted submission tables were added or changed.
