# License Decision Required

No license has been granted by this draft release package. Before publishing to GitHub or Zenodo, every rights holder must approve a license compatible with the intended reuse.

Recommended decision points:

- **Code:** MIT or BSD-3-Clause are common permissive software licenses.
- **Documentation and aggregate tables:** CC-BY-4.0 is common when attribution is desired.
- **Bundled third-party content:** do not include it unless redistribution is permitted by its original license or terms.

After approval, add the selected license text as `LICENSE`, replace the pending license strings in `CITATION.cff` and `.zenodo.json`, and state the choice in the GitHub release notes.
